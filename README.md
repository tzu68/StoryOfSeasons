我想要建立一個 "牧場物語 風之市集" 的攻略網站，包含以下分頁與功能：

1. 首頁
   內容：快速連結到其他分頁。
   功能：
   使用導航列或按鈕清單，連結到以下分頁：
   行事曆
   居民資訊
   動物外觀
   礦物分布
2. 行事曆
   資料來源：
   節日名稱與日期：Event.txt
   居民生日清單：people.txt
   功能：
   互動式行事曆，允許用戶切換月份。
   節日的日期需要根據年份進行即時運算（例如某個周四）。
   節日與居民生日以不同顏色標記。
3. 居民資訊
   資料來源：people.txt
   功能：
   每個居民的資訊以卡片（card）的形式展示。
   卡片內容包括：姓名、生日、喜好/討厭的物品。
   使用 CSS 美化卡片樣式。
4. 動物外觀
   圖片來源：images/animals/
   功能：
   圖片展示，無需額外文字。
   使用 CSS Grid 或 Flexbox 排列圖片。
   支援左右滑動瀏覽圖片。

5. 礦物分布
   資料來源：
   春季：images/stone/spring*\*
   夏季：images/stone/summer*\_
   秋季：images/stone/fall\_\_
   冬季：images/stone/winter\_\*
   功能：
   圖片展示，按季節分類。
   使用分頁或標籤（tabs）切換季節。

請幫我設計這個網站的 HTML 結構、CSS 樣式，以及必要的 JavaScript 程式碼，實現上述功能。

需要支援手機板的使用。
